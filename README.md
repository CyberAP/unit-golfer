
helps shorten pixel units for cssbattle.dev

# usage

```
node unit-golf 325

[ '344q',
  '325px',
  '108vh',
  '8.6cm',
  '245pt',
  '3.39in',
  '20.3em',
  '45.3ex',
  '81.3vw',
  '86.3mm',
  '20.3pc' ]
```

you can adjust the precision too:

```
node unit-golf 325 2

[ '20pc',
  '340q',
  '86mm',
  '81vw',
  '45ex',
  '20em',
  '8.6cm',
  '240pt',
  '3.4in',
  '110vh',
  '330px' ]
```
